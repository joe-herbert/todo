# todo
A Bash CLI for keeping a todo list  

## Installation
[todo](todo) is a bash script, so you can just download it somewhere and make sure it's added to your `$PATH`.  
For example, this one liner:  

    $ sudo sh -c "curl https://raw.githubusercontent.com/joe-herbert/todo/main/todo -o /usr/local/bin/todo && chmod +x /usr/local/bin/todo"
